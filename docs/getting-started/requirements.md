# Requirements

Before proceeding, please make sure you have **at least** met the following prerequisites:

| Requirement  | Version       |
|--------------|---------------|
| PHP          | `^8.1`        |
| Laravel      | `^9.0, ^10.0` |
| Livewire     | `^2.0`        |
| Alpine.js    | `^3.0`        |
| Tailwind CSS | `^3.0`        |
